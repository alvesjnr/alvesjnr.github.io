Simulating Natural Systems - Part 0

Sat 21 December 2013
By Antonio Ribeiro

In Simulation.

tags: simulationprogramming

I am a really fan of computer simulations. I think that this is a fantastic (and really broad) field in computer science.

In my Msc. in Computer Science I spent a long time studding discrete events simulation. Currently I'm working quite close to numerical simulations, using techniques like finite-difference time-domain and eigenmode expansion to simulate the behavior of electrodynamics waves in dielectric media (like glass, air and silicon).

One day, in one of those forums (like reddit), I discovered The Nature of Code, a very nice book for someone who is aiming to to debut in simulations. This book, written by Daniel Shiffman, brings a lot of interesting concepts about tools to model natural systems (like vectors, force, oscillation) and link them to more elaborated concepts, like cellular automata and neural networks, to simulate complex systems.

The major drawback of this book, from my perspective, is that it uses Java as the language to code the examples. Although Java is a very powerful language, it is far from been the favorite language for hobbyists and pet projects. Due to this I proposed to implement the examples and exercises from the book in a mix o C and Python (working with C/Python integration is my not-so-secret agenda in this project). All the code I'll put in this dedicated Github repository.

I don't have any particular schedule to update this project, which means that sometimes I'll post twice about this in the same week and sometimes I will spend two whole months not talking about this. If you are interested I'll suggest you to follow the project on Github.

I appreciate your curiosity :)
